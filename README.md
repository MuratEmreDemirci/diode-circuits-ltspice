# Diode Circuits Simulation (LTspice)

## 📌 Overview
This project includes the design and simulation of fundamental diode-based circuits using LTspice. The aim is to analyze signal transformation, clipping, and voltage regulation behaviors.

## ⚙️ Technologies
- LTspice
- Analog Circuit Design

---

## 🔌 Full-Wave Rectifier
- Bridge rectifier using 4 diodes
- Converts AC to DC (full-wave)

---

## ✂️ Clipper Circuits
- Positive clipper (~5.7V)
- Dual clipper (+13V / -5V)
- Asymmetric clipper (+4.7V / -5.7V)

---

## 🔋 Zener Voltage Regulator
- Output stabilized despite input changes
- Demonstrates voltage limiting

---

## 📊 Results
Simulation results show waveform shaping, clipping behavior, and voltage regulation.
